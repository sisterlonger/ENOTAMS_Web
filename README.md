# TinyPro of vue

## 说明

此项目为 tiny-toolkit-pro 套件初始化的以 TinyPro of vue 为模板的一个 Vue 项目

## 环境要求

- **Node.js**: 24.x LTS（推荐 24.16.0，见 `.nvmrc` / `.node-version`）
- **npm**: 10+

使用 nvm / fnm / Volta 时，进入项目目录后会自动读取 `.nvmrc` 或 `.node-version`。

```bash
# nvm
nvm install
nvm use

# fnm
fnm use

# Volta（若已安装）
volta install node@24
```

## 用法

```bash
npm install
npm run start
npm run build
```
